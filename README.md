## üìñ About The Project

**NeonFocus Ultra** is a performance-optimized web application designed to help developers and typists improve their efficiency. Built with **Vanilla JavaScript**, it avoids framework bloat to deliver millisecond-accurate timing and smooth 60fps animations.

### ‚ú® Key Features
* **üé® Cyberpunk Glassmorphism:** A fully responsive UI with neon accents (`#00f2ff`, `#7000ff`), animated backgrounds, and dynamic glows.
* **üß† Smart Analysis Engine:** Tracks every keystroke to calculate real-time WPM and Accuracy.
* **üìâ Weak Point Detection:** Post-session algorithm identifies your top 5 "worst keys" so you know exactly what to practice.
* **‚ö° Zero-Latency Typing:** Uses `DocumentFragment` for batch DOM insertion, ensuring no input lag even on lower-end devices.
* **üéÆ Dynamic Difficulty:**
    * **Easy:** Simple vocabulary for warm-ups.
    * **Medium:** Complex sentence structures with punctuation.
    * **Hard:** Scientific and technical jargon to test mastery.
